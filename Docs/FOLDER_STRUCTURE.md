# 📁 Plugin Folder Structure

```
audio-waveform-player/
├── 📄 main.js                    # Compiled plugin code (REQUIRED)
├── 📄 manifest.json              # Plugin metadata (REQUIRED)  
├── 📄 styles.css                 # Plugin styling (REQUIRED)
├── 📄 versions.json              # Version compatibility
├── 📄 README.md                  # Complete documentation
├── 📄 INSTALLATION.md            # Installation guide
├── 📄 Audio Player Test.md       # Test note with examples
├── 🎵 lick 1-guitar-120.wav      # Sample audio file
└── 🎵 lick 1-no-guitar-120.wav   # Sample audio file
```

## 🎯 What You Need to Install

**Minimum Required Files:**
- `main.js`
- `manifest.json` 
- `styles.css`

**Optional but Recommended:**
- `versions.json` (for compatibility)
- `README.md` (for reference)
- Sample audio files (for testing)

## 📍 Installation Location

Copy this entire folder to:
```
YourVault/.obsidian/plugins/audio-waveform-player/
```

**Example full path:**
```
/Users/yourname/Documents/MyVault/.obsidian/plugins/audio-waveform-player/
```

## ✅ Verification

After copying, your plugin folder should contain at minimum:
- ✅ main.js
- ✅ manifest.json  
- ✅ styles.css

Then enable the plugin in: **Settings > Community Plugins > Audio Waveform Player**
